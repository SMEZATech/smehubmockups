Engine Architecture

Native Canvas Renderer

Custom CanvasRenderingContext2D engine — exact measureText(), greedy word wrap with character-level overflow fallback, composite-op tinting, and per-shape DPR scaling for crispness at any resolution.

Inline Web Worker

Worker spawned from URL.createObjectURL(new Blob), runs all 6 slide renders on OffscreenCanvas, ZIPs via JSZip, returns one blob. Main thread never blocks.

Binary-Search Auto-Fit

For each headline: fitFontSize(W, H) binary-searches the largest font size where wrapped lines fit both width AND height. ~9 iterations, sub-ms per call.

Smart Article Slicer

Sentence-aware bin-packing across 4 carousel body slides. Lookbehind regex on [.!?], greedy fill, never splits sentences mid-way.
